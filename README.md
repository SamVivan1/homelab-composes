# homelab-composes

A collection of Docker Compose configurations for self-hosted services and applications in a home lab environment.

## Features

- Pre-configured Docker Compose files for popular services
- Easy setup and deployment
- Service documentation and usage examples

## Quick Start

1. Clone the repository
2. Navigate to the desired service directory
3. Review the `.env` file and adjust configuration as needed
4. Run `docker-compose up -d`
